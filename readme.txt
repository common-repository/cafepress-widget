The CafePress Widget allows users to pull products from their CafePress store and advertise them on their WordPress site. 

== Description ==

The CafePress Widget (http://www.cafepress.com/) was created to answer the call of many shopkeepers: I want a fun way to advertise my gear.  We've finally created the perfect tool for shopkeepers and affiliates alike to promote their merchandise and generate sales.  You can use it to display your favorite goodies for fun as well!
 
Features: 

* Free and extremely easy to install and use  
* User and search engine friendly
* Incorporates all products into a fluid stream

== Installation ==

To install the plugin, please follow these steps:

1. Download the zipped plugin file to your local machine.
2. Unzip the file.
3. Upload the *'cafepress-widget'* folder to the *'/wp-content/plugins/'* directory on your webserver.
4. Activate the plugin through the 'Plugins' menu in WordPress.
5. Go to 'Widgets' section under 'Appearance' menu in WordPress and add the *'CafePress Widget'* in your sidebar.
6. Open the widget options and...
   - Specify the Widget Title
   - Select a Color Theme
   - Enter products to be displayed
